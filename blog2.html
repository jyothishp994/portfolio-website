  • Pattern Printing

    August, 2020

    Pattern printing is something which is really interesting irrespective of the programming language. Let's see how we can print the below pattern

    *****
    ****
    ***
    **
    *

    The best way to think about pattern printing is to put up the required pattern in terms of a two dimensional array and find out the relation with vertical and horizontal dimension. Let's see below.

    J i01234
    0*****
    1 ****
    2 ***
    3**
    4 *

    Let's consider horizontal index as i and vertical index as j. So we need two loops which are nested. Loop with i to go line by line and Loop with j to print the * at [i][j] position in the matrix. If we look at relation between i and j, it is clear that * needs to be printed untill j<=(n-i) holds true, where n is the length of * in first line. ie here n=4. Let's check for a few iterations.

    When i=0, j=0 to 4 => Print * untill 0<=4 => *****
    When i=1, j=0 to 3 => Print * untill 0<=3 => ****
    When i=2, j=0 to 2 => Print * untill 0<=2 => ***
    When i=3, j=0 to 1 => Print * untill 0<=1 => **
    When i=3, j=0 to 0 => Print * untill 0<=0 => *
